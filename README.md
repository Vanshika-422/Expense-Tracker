**Developed by:** Vanshika & Muskaan

## 📋 Project Overview
A robust Java-based expense management system featuring a Graphical User Interface (GUI), persistent file storage, and data analytics.

### 🛠️ Key Features
* **Data Persistence:** Automatically saves and loads data using `FileHandler`.
* **Smart Analytics:** Logic to calculate total spending while filtering out upcoming bills.
* **Calendar View:** Search functionality to find payments and bills for specific dates.
* **Interactive GUI:** Built with Java Swing for a user-friendly experience.

### 👩‍💻 Role Distribution
* **Muskaan (System Architect):** Developed the `Expense` blueprint, `ExpenseManager` logic, and the `FileHandler` for data storage.
* **Vanshika (Data Analyst):** Developed the `ExpenseAnalyst` engine for math logic, category filtering, and the user-facing GUI integration.
